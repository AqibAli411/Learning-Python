# Variables & Data Types

## Main types:
- int = whole numbers
- float = decimals  
- str = text (strings)
- bool = True/False

## Important stuff:
- dont need to declare type, python figures it out
- can change variable type anytime (dynamic typing?)
- use type() to check what type something is
- int(), float(), str() to convert between types
