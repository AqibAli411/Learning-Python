# String Methods

so many methods for strings!!

## useful ones:
- .upper() .lower() .capitalize() .title()
- .replace(old,new)
- .split() - turns string into list
- .strip() - removes whitespace
- len() - get length

## indexing:
- starts at 0
- negative numbers count from end
- slicing with [start:end]

## f-strings:
- put f before the quote
- use {variable} inside string
- way easier than concatenating with +
