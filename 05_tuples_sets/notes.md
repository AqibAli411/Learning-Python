# Tuples and Sets

## Tuples:
- like lists but IMMUTABLE (cant change)
- use parentheses ()
- faster than lists
- can unpack them into variables

when to use:
- data that shouldnt change
- returning multiple values from function
- dictionary keys

## Sets:
- unordered collection
- NO DUPLICATES
- use curly braces {}
- good for removing duplicates from list

operations:
- union (|) - combine
- intersection (&) - common elements
- difference (-) - in first not second
