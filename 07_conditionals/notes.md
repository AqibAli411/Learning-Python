# Conditionals

if/elif/else for making decisions

## comparison operators:
- == equal
- != not equal
- > greater than
- < less than
- >= greater or equal
- <= less or equal

## logical operators:
- and - both must be true
- or - at least one must be true
- not - opposite

## important:
- INDENTATION MATTERS!! (learned this the hard way)
- use elif not "else if"
- can nest if statements
- empty lists/strings/0 are "falsy"

## ternary operator:
`value = x if condition else y`
