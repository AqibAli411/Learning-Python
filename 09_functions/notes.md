# Functions

reusable code blocks!

## syntax:
```
def function_name(parameters):
    # code
    return value
```

## key concepts:
- def keyword to define
- parameters = inputs
- return = output
- can have default parameter values
- can return multiple values (as tuple)

## special parameters:
- *args = variable number of arguments
- **kwargs = variable number of keyword arguments

## lambda functions:
- one-line anonymous functions
- `lambda x: x * 2`
- useful with map(), filter(), sorted()

## scope:
- variables inside function are local
- use global keyword to modify global variables (but try to avoid)
